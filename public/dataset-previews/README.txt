Dataset Preview Images

This directory should contain the following image files for the dataset cards:
- demographics.png: Visualization for Census Demographics dataset
- medical.png: Visualization for Medical Research dataset
- financial.png: Visualization for Financial Transactions dataset
- iot.png: Visualization for IoT Sensor Readings dataset

If these images are not available, the UI will fall back to using the 
dataset-placeholder.png file in the parent directory.

For production, replace these placeholders with actual data visualizations
that represent each dataset with appropriate charts, graphs, or infographics.
